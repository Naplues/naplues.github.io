# 开工当天
✅ 需要确认
核对开工图纸：平面布局、拆改位置、层高、门洞尺寸，和合同效果图一致
物业手续齐全：装修押金、垃圾清运、施工时段、邻里告知
现场保护：入户门、电梯、强弱电箱、下水管道全包保护
确认现场负责人、监理、设计师三方对接群，每日报备进度
❌ 防坑重点
口头承诺增项、加钱一概不认，所有变更必须书面签字
禁止当天额外预收工程款、杂费
严禁开工乱砸承重墙、配重墙、飘窗配重


# 铲墙 / PC 放样
✅ 需要确认
铲墙：铲除原开发商劣质腻子、空鼓墙皮，墙面坚实无粉化
PC 实景放样：地面 / 墙面弹出家具、柜体、水电、门洞、吊顶轮廓，一比一还原
核对放样：衣柜尺寸、床头位置、开关插座点位、背景墙造型
❌ 防坑重点
偷工：只局部铲墙，后期墙面起皮开裂
放样敷衍、不弹线，后期柜子装不上、插座被遮挡
后期借 “墙面找平、补铲墙” 恶意加钱


# 泥工进场砌墙
✅ 需要确认
新旧墙体连接处植筋加固，防开裂
卫生间、厨房新墙底部做混凝土止水坎（防水导梁）
墙体垂直、平整，门洞上方加过梁
红砖 / 轻质砖材料和合同标注一致
❌ 防坑重点
新墙不做止水坎，后期墙面返潮渗水
不加过梁，门洞下沉开裂
垃圾墙体、劣质材料偷换


# 电定位（全屋关键）
✅ 需要三方到场：你 + 设计师 + 水电工
逐个确认：灯位、开关、普通插座、空调、冰箱、洗碗机、热水器、扫地机基站、智能马桶点位
区分回路：照明 / 普通插座 / 厨房 / 卫生间 / 空调独立分开
确认电线平方：
照明 1.5²、普通插座 2.5²、厨房 / 热水器 / 空调 4²、入户主线 6²
❌ 防坑重点
点位少预留、后期家电没地方插
大功率电器不给独立回路、混用线路
口头答应高配，实际施工偷换细线


# 水电施工（隐蔽工程，重中之重）
✅ 需要确认
水管：走顶不走地，左热右冷、上热下冷，横平竖直
电线：强弱电分管分槽，间距≥30cm，不共管不共盒
厨卫水管保温棉包裹，防冷凝水
所有暗盒水平对齐，固定牢固
❌ 防坑重点
水电乱开槽、切断钢筋、地面大面积乱刨
电线不分色、非国标线、4² 偷换成 2.5²
水管接头劣质、不固定，后期漏水泡家
按米额外加价（全包必须一口价，无增项）


# 水电验收（必须亲自到场）
✅ 需要确认
水管打压测试：0.8MPa 保压 30 分钟，不掉压、不渗水
通电通断测试，所有回路漏电保护正常
拍照 + 录像留存全屋水电走向图，纸质存档
核对电线平方、品牌、点位和定位一致
❌ 防坑重点
不打压、假打压直接封槽
隐蔽处私自改线、减配线材
验收不签字就偷偷封水泥


# 空调进场安装（中央空调 / 挂机）
✅ 需要确认
内机水平安装，排水坡度达标，防积水倒灌
铜管保温包扎严实，风口尺寸和吊顶匹配
电路单独 4²/6² 专线，接地保护
防尘保护，防止后续装修进灰
❌ 防坑重点
排水无坡度，后期漏水泡吊顶
铜管偷短、不加保温，结露发霉
电路共用普通插座，频繁跳闸


# 泥工进场贴砖
✅ 需要确认
先做防水→48 小时闭水试验（通知楼下确认无渗漏）
厨卫墙压地工艺，地漏做八字角、排水坡度合理
瓷砖泡水处理、铺贴平整，缝隙均匀
窗台石、门槛石安装到位
❌ 防坑重点
防水不做闭水、偷减涂刷遍数
瓷砖空鼓（敲击检查，空鼓率≤5%）
地面无坡度、卫生间积水不走
色差砖、破损砖偷偷铺贴


# 木工进场定位
✅ 需要确认
吊顶、双眼皮、边吊、背景墙基层精准定位
定制衣柜、鞋柜、餐边柜尺寸复核，避开开关水管
空调出风口、检修口预留标准位置
❌ 防坑重点
吊顶尺寸不合理，压抑、藏灰
柜子和水电冲突，后期开孔破坏管线
不预留空调检修口，后期维修砸顶


# 木工施工
✅ 需要确认
轻钢龙骨 / 实木龙骨间距标准，加固防开裂
石膏板 V 型拼缝，防开裂处理
厨卫吊顶耐潮材料，防锈螺丝点漆
❌ 防坑重点
龙骨稀疏、不加固，后期吊顶下沉开裂
用劣质受潮板材，发霉变形
偷减辅材、省钉子省龙骨


# 泥木工联合验收
✅ 需要确认
瓷砖空鼓、平整度、阴阳角 90° 顺直
吊顶牢固、无松动、造型和图纸一致
厨卫排水通畅、无积水
现场卫生清理，成品保护做好
❌ 防坑重点
掩盖空鼓、瑕疵，直接进入下一道工序
尺寸偏差不整改，后期定制家具装不进去


# 油漆进场
✅ 需要确认
墙面挂网防开裂（新旧墙、顶地交接必挂）
批刮 2–3 遍腻子，每遍干透再打磨
底漆 1 遍、面漆 2 遍，调色漆先试色再全屋刷
❌ 防坑重点
腻子不干直接刷漆，后期发霉起皮
少刷一遍漆、兑水过多，遮盖力差
名牌漆偷换成低端系列、假货


# 安装工程（主材集成安装）
包含：室内门、橱柜、衣柜、地板、卫浴、开关灯具、五金✅ 需要确认
全屋木门缝隙均匀、开关静音、密封条完好
橱柜五金开合顺畅，台面无缝不渗水
地板找平达标、无异响、防潮处理
卫浴试水不漏水、下水密封
❌ 防坑重点
安装磕碰不修补、成品无保护
五金减配、劣质铰链拉手
台面打胶发霉、漏水返味


# 油漆修补
✅ 需要确认
安装磕碰、钉眼、缝隙统一修补找平
全屋阴阳角顺直，无色差、刷痕、流挂
柜子周边、门窗收口细腻美观
❌ 防坑重点
敷衍修补、局部色差不处理
收口粗糙，后期藏灰难打理


# 整体竣工验收
✅ 需要三方到场：你 + 监理 + 装修公司逐项核查：水电、防水、墙地顶、门窗、柜体、卫浴、电器功能不合格项列明清单，限期免费整改，不整改不结尾款
❌ 防坑重点
催促快速签字验收、隐瞒瑕疵
口头承诺后期修补，不认账
未整改强行索要全款


# 16、开荒保洁
✅ 需要确认
清除漆点、胶印、水泥残渣、玻璃清洁、柜体除尘精细保洁，不划伤瓷砖、五金、漆面
❌ 防坑重点
粗暴清洁、强酸腐蚀石材五金
表面敷衍打扫，死角不清理


# 17、竣工交付
✅ 需要确认
移交：水电竣工图、材料合格证、保修卡、售后联系方式
核对质保：水电 / 防水≥5 年，硬装≥2 年
结清款项：预留 5%–10% 质保金，期满无问题再付
❌ 防坑重点
不交售后资料、隐瞒保修责任
强制结清所有尾款，不留质保保障
